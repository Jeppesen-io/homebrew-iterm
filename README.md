# homebrew-iterm
### Wraps `iterm2` cask
iTerm, for me! Nothing crazy, just lots of little tweaks.

* True color support
* Minimal boders
* Clean, bright color scheme
* CTRL-left or right to move cursor in word-direction
* Lots of other adjustments I've long forgoten
